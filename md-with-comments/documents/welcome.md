# Welcome to MDComments

This is the short sample document. Select any passage of text with your mouse and an
**Add comment** button will appear, just like in a word processor.

## What you can do

- Attach a comment to any text range
- Reply to comments to form a thread
- Edit, resolve, and delete comments
- Reload the page — everything persists

## A small table

| Action | Where |
|--------|-------|
| Comment | Select text in the document |
| Reply | Use the card in the margin |

## Some code

```js
function greet(name) {
  return `Hello, ${name}!`;
}
```

> Comments live in a sidecar JSON file next to the markdown document, so the
> source file itself is never modified.
